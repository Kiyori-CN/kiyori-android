# 数据安全说明

## B站登录数据加密

### 加密方式
- AES-256 加密存储
- 密钥由 Android KeyStore 管理
- 硬件安全模块保护，应用无法导出

### 数据用途
- 仅用于调用 B站 API
- 不上传至第三方服务器
- 完全保存在本地设备

### 用户控制
- 可随时退出登录
- 退出后本地数据将被清除
- 应用卸载后数据自动销毁

详细技术实现：[B站登录安全分析](bilibili_security_analysis.md)

---

## 本地数据存储

### 存储内容
- 视频、字幕、弹幕文件：用户指定文件夹
- 播放进度：本地数据库
- 登录凭证：加密存储

### 存储特性
- 不同步或备份到云端
- 用户完全控制存储位置
- 可随时查看、管理或删除

---

## 应用权限

### 存储权限（管理所有文件）
用于读取本地视频、保存字幕弹幕、视频截图、下载文件

### 网络权限
用于在线播放、弹幕匹配下载、视频下载、WebDAV 访问

---

## 数据流向

### 本地数据
```
用户设备
├─ 视频/字幕/弹幕文件（本地存储）
├─ 播放进度（本地数据库）
└─ 登录凭证（加密存储）
```

### 网络请求
```
应用 → B站 API（番剧功能）
应用 → 弹弹play API（弹幕匹配）
应用 → WebDAV 服务器（用户配置）
```

所有网络请求仅在使用对应功能时发起，不包含数据收集或上传。
