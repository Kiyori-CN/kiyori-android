# 数据安全说明

本文档详细说明应用的数据安全保护措施。

---

## 🔐 B站登录功能安全

### 加密存储

- 登录凭证使用 **AES-256 加密**存储在本地
- 加密密钥由 Android KeyStore 系统管理
- KeyStore 密钥受硬件安全模块保护，**应用无法导出**

详细的安全实现请参阅：[B站登录安全分析](bilibili_security_analysis.md)

### 数据用途

- 登录信息**仅用于调用B站API**
- **不会上传**到任何第三方服务器
- 数据完全保存在用户本地设备

### 用户控制

- 用户可随时在设置中**一键退出登录**
- 退出登录将**彻底清除**所有本地存储的登录数据
- 应用卸载后，所有登录数据将**自动永久销毁**

---

## 📦 弹幕与视频数据

### 数据存储位置

- 弹幕和视频文件保存在**用户指定的本地文件夹**
- 用户可完全控制存储位置
- 数据仅存储在本地，**不会同步或备份到云端**

### 下载机制

- 下载功能完全由**用户主动触发**
- 应用不会自动下载任何内容
- 用户可随时查看、管理或删除已下载的文件

### 重要提示

⚠️ **视频下载功能仅供个人学习使用，严禁商业用途**

- 下载内容的**版权归原作者所有**
- 使用者需遵守相关法律法规及平台服务协议
- 法律责任由用户自行承担，与本项目无关

---

## 🔑 应用权限说明

本应用仅请求以下必要权限：

### 存储权限（管理所有文件）

**用途**：
- 读取本地视频文件进行播放
- 保存字幕、弹幕文件
- 保存视频截图
- 保存下载的视频/弹幕文件

**说明**：此权限为播放器核心功能所必需，不会访问与应用功能无关的文件。

### 网络权限

**用途**：
- B站番剧在线播放
- 弹幕匹配和下载
- 视频下载（用户主动触发）
- WebDAV 云端视频访问

**说明**：网络请求仅在用户主动使用相关功能时发起，不会进行后台数据收集或上传。

## 🛡️ 隐私保护承诺

### 我们不做的事

- ❌ **不收集**用户的任何个人信息
- ❌ **不上传**用户数据到我们的服务器（我们根本没有服务器）
- ❌ **不分享**用户数据给任何第三方
- ❌ **不追踪**用户行为或使用习惯
- ❌ **不包含**任何广告或数据分析 SDK

### 我们做的事

- ✅ 所有功能均在**本地设备**上运行
- ✅ 项目**完全开源**，所有代码公开可审查
- ✅ 使用业界标准的**加密算法**保护敏感数据
- ✅ 遵守 **GPL-3.0** 开源协议

## 🔍 数据流向说明

### 本地数据

```
用户设备
  ├─ 视频文件 (用户本地存储)
  ├─ 字幕文件 (用户本地存储)
  ├─ 弹幕文件 (用户本地存储)
  ├─ 播放进度 (本地数据库)
  └─ 登录凭证 (加密存储在本地)
```

### 网络请求

```
应用 → B站API (仅在使用B站功能时)
     ├─ 番剧信息获取
     ├─ 视频播放地址
     └─ 弹幕下载

应用 → 弹弹play API (仅在使用弹幕匹配时)
     ├─ 视频文件匹配
     └─ 弹幕数据下载

应用 → WebDAV 服务器 (仅在用户配置后)
     └─ 云端视频文件访问
```

**重要**：所有网络请求均为功能所需，不包含任何用户数据收集或上传行为。
